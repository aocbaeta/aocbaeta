## Featured Project: Honeycomb BdG Quantum

A scientific computing project for studying superconductivity and spin-orbit coupling in honeycomb lattices using the Bogoliubov-de Gennes formalism.

The project implements an 8x8 BdG Hamiltonian with sublattice, spin, and Nambu degrees of freedom, including s-wave and p+ip superconducting pairing, intrinsic spin-orbit coupling, Rashba interaction, minimum-gap analysis, and phase-map generation.

It also explores reduced quantum-computing mappings with PennyLane, Qiskit, and Cirq, connecting classical numerical simulation with quantum algorithms such as VQE.

**Technologies:** Python, NumPy, SciPy, Matplotlib, SymPy, PennyLane, Qiskit, Cirq.
